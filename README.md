# Rock Paper Scissors

In a distant and deeply inconvenient future, the Dark AI has seized control of the console and challenged humanity to an ancient ritual duel. Your mission is simple: survive 5 rounds of Rock, Paper, Scissors, outplay the machine, and remind this smug digital tyrant that flesh-and-blood instincts still matter.

Simple browser-based Rock, Paper, Scissors game played over 5 rounds against the computer.

## What You Need

- A web browser
- We recommend using `Firefox`
- The project files, including `index.html` and `script.js`
- Access to the browser console, because the game prints the results there

## How to Start

1. Open `index.html` in your web browser.
2. The game starts automatically and shows an alert about the Dark AI.
3. Open the browser console to follow the battle logs and score.
4. Respond to the prompts to play each round.

## What to Enter During the Game

- When a prompt appears, enter one of these choices: `Rock`, `Paper`, or `Scissors`
- Your input is accepted even if you use different letter casing, such as `rock` or `SCISSORS`
- If your input is not valid, the game asks you to try again
- If you cancel the prompt, the game asks whether you want to surrender

## How to Know the Result

- You won a round if the game shows `You Won!`
- You lost a round if the game shows `You Lose!`
- You drew a round if the game shows `It is a tie!`
- After the 5 rounds, the final score in the console shows whether you beat the computer, lost to it, or finished in a draw
- If you surrender, the game ends immediately and the computer wins
